# Conquest

Демонстрацию верстки можно посмотреть [по этой ссылке](https://jinv.ru/conquest/).

Верстка сделана адаптивной, оптимально отображается на всех разрешениях от 360px до 1920px. 

Для автоматической перезагрузки страницы браузера я воспользовался сборщиком Parcel.

Проект нормально работает если используется сервер, будь то сервер сборщика Parcel, либо GitHub Page. Если же открыть файл index.html в файловой системе, то браузер не будет отображать иконки SVG.
